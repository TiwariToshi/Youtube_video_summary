<h2>Project -Youtube Video Transcribe Summarizer LLM App</h1>
<h2>Project Overview<h2>
<h3>It is a LLM based application that summarizes the youtube video/podcasts </n>
  and helps to go through the relevant content.
</h3>

<h3>Implementation</h3>
<p>1.Youtube provides YouTubeTranscriptApi that is used to transcript of the video. </p>
<p>2.Gemini LLM takes that raw transcript and performs intelligent tasks</n>
<ol><p>i.Short summary</p> </n>
<p>ii.Bullet points </p></n>
<p>iii.Key takeaways </p> </n>
</ol>
</p>
<p>3.Streamlit is used for UI</p>
